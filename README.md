# FitAppTracker 💪

A full-stack fitness tracking web application built with React and Node.js.

## Features
- Track workouts
- Add exercises
- View progress
- Responsive UI

## Tech Stack
- React (Vite)
- Node.js + Express
- Tailwind CSS

## Live Demo
https://yourusername.github.io/fitapp

## Installation
cd client
npm install
npm run dev
